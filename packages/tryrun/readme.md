# [tryrun] ⚠️ WIP ⚠️

Compose programs with errors and dependencies tracked at the type level. Zero runtime surprises.

    npm add tryrun

## Features

- Tiny, simple API
- Written in TypeScript with full type safety
- Tracked errors and requirements
- Dependency injection
- Zero dependencies
